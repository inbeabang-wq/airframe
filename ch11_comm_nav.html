무선 항법

Radio Navigation📄 PDF p.604

주요 무선 항법 장비입니다.

1) VOR — 방위각(Radial) 기준 항법

2) ILS — 정밀 접근(로컬라이저+글라이드슬로프)

3) DME — 거리 측정

4) GPS — 위성 기반 위치

각 장비는 안테나 설치 위치와 접지(Bonding)가 중요합니다.

📖 핸드북 정리 (FAA-H-8083-31B) ▼
Main radio navigation systems. 1) VOR — radial-based navigation 2) ILS — precision approach (localizer + glideslope) 3) DME — distance measuring 4) GPS — satellite-based position Antenna placement and bonding are important for each.
주요 무선 항법 장비입니다. 1) VOR — 방위각(Radial) 기준 항법 2) ILS — 정밀 접근(로컬라이저+글라이드슬로프) 3) DME — 거리 측정 4) GPS — 위성 기반 위치 각 장비는 안테나 설치 위치와 접지(Bonding)가 중요합니다.

전파 간섭 저감

Reducing Interference📄 PDF p.604

무선 간섭 저감 방법입니다.

1) 차폐(Shielding) — 전선을 금속 차폐로 감쌈

2) 접지·본딩(Bonding) — 전위차 제거

3) 정전기 방전 위크(Static Discharge Wick) — 정전기 배출

부적절한 본딩은 잡음과 낙뢰 손상을 유발합니다.

📖 핸드북 정리 (FAA-H-8083-31B) ▼
Methods to reduce radio interference. 1) Shielding — wrap wires in metal shielding 2) Bonding — eliminate potential differences 3) Static discharge wicks — bleed off static Poor bonding causes noise and lightning damage.
무선 간섭 저감 방법입니다. 1) 차폐(Shielding) — 전선을 금속 차폐로 감쌈 2) 접지·본딩(Bonding) — 전위차 제거 3) 정전기 방전 위크(Static Discharge Wick) — 정전기 배출 부적절한 본딩은 잡음과 낙뢰 손상을 유발합니다.

C11Q1정밀 접근에 로컬라이저와 글라이드슬로프를 함께 사용하는 계통은?Which system uses both a localizer and glideslope for precision approach?
  • A. VORVOR
  • B. DMEDME
  • C. ILSILS
  • D. GPSGPS
정답: C — ILS
ILS는 로컬라이저(방위)와 글라이드슬로프(강하각)로 정밀 접근을 제공합니다.ILS provides precision approach with a localizer (course) and glideslope (descent).
