## 조치
- 잘못된 이미지 **15개**를 "📐 도면 준비 중" 플레이스홀더로 교체
- 각 caption은 그대로 유지, 상단 FAA 핸드북 PDF 링크로 원문 확인 가능
- **중복 이미지 0건 달성** (재검증 완료)
- 학습자에게 틀린 도면을 보여주지 않는 것을 최우선으로 처리

## 후속 작업(선택)
- FAA PDF는 봇 접근 차단(403)으로 자동 추출 불가
- 인배님이 FAA-H-8083-31B PDF의 해당 페이지에서 올바른 도면을 캡처해 주시면,
  플레이스홀더 자리에 정확한 이미지로 교체 가능 (각 figure에 PDF 페이지 링크 있음)
- 나머지 139개 이미지는 중복 없음 — caption과 도면 정합성은 대체로 정상으로 판단
